# Research — PsyFusions

## Позиционирование

**Ключевое сообщение:** Психолог и гипнолог, который работает с корнем проблемы, а не симптомами — через подсознание, мягко и конфиденциально.

**Дифференциация:**
- Интегративный подход (гипноз + арт-терапия + коучинг) — редкое сочетание
- Работает в Европе и Азии — понимает контекст эмигрантской жизни
- Акцент на безопасности и конфиденциальности — важно для аудитории с табу вокруг психологии
- Бесплатная первая встреча — низкий барьер входа

---

## Сильные стороны текущего сайта

### Дизайн и визуальная подача
- Современная, минималистичная эстетика — вызывает доверие
- Цветовая палитра (тёмно-зелёный + терракота + кремовый) — тёплая, профессиональная
- Плавные анимации (Framer Motion) — воспринимается как премиальный продукт
- Адаптивная верстка — работает на мобильных
- Декоративные blob-фоны — создают мягкость без кричащего дизайна

### Структура и контент
- Логичная последовательность: методика → диагностика → запросы → о себе → процесс → отзывы → контакты
- Квиз-диагностика — мощный инструмент вовлечения, даёт ценность до продажи
- FAQ с ответом на главный страх («гипноз — это не магия»)
- Секция с дипломами — повышает доверие
- Чёткие CTA: «Бесплатная встреча 15 мин» и «Записаться»

### Технические плюсы
- React + Vite — быстрый и современный стек
- Tailwind CSS — быстрое изменение стилей
- Express API для форм — централизованная обработка заявок

## Слабые стороны и зоны роста

### Контент
- **Нет реального фото** — placeholder-изображения из picsum сильно снижают доверие
- **Нет реальных отзывов** — тексты звучат шаблонно, нет фото клиентов
- **Нет реальных дипломов** — изображения заглушки
- **Нет цен** — аудитория не понимает, сколько это стоит (создаёт тревогу)
- **Нет конкретных кейсов** — «5 сессий и выгорание прошло» не убеждает без деталей

### Конверсия
- **Форма отправляет на localhost** — в production нет реальной интеграции (email/Telegram бот)
- **Нет social proof сверху** — звёзды и отзывы появляются слишком поздно
- **Нет таймера срочности** — «2 места на этой неделе» без визуального усиления

### SEO и доступность
- Нет мета-описания (решено в index.html)
- Нет структурированных данных (Schema.org для специалиста)
- Изображения без alt-текстов (частично)
- lang="en" в оригинале — исправлено на ru

### Технические
- Зависимость от CDN picsum для всех изображений
- Нет реальной отправки форм (только console.log в server.ts)
- Нет `.env` файла с реальными токенами

## Рекомендации по улучшению

### Приоритет 1 — Критично для доверия
1. **Добавить реальное фото** Оксаны на Hero и в About
2. **Добавить реальные отзывы** с именами и желательно фото (с разрешения)
3. **Подключить реальную отправку форм** — Telegram бот или email через nodemailer
4. **Добавить реальные фото дипломов**

### Приоритет 2 — Конверсия
5. **Добавить блок с ценами** (или хотя бы «от N €/сессия»)
6. **Добавить кнопку-стикер WhatsApp** как второй floating CTA
7. **Усилить hero-цитату** — добавить первое имя реального клиента

### Приоритет 3 — Рост
8. **Добавить видео-приветствие** — 30-60 секунд от Оксаны повысят конверсию x2
9. **Добавить блог/заметки** — для SEO и прогрева аудитории
10. **Яндекс.Метрика или Google Analytics** — понять откуда приходят клиенты
11. **Schema.org разметка** для специалиста (MedicalBusiness или Person)

## Конкурентный контекст

Сильные конкуренты в нише «онлайн психолог для русскоязычных за рубежом»:
- Используют более стандартные дизайны (шаблонный WordPress/Tilda)
- PsyFusions выигрывает по дизайну и современности стека
- Слабое место — нет социального доказательства и реального контента
